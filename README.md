# procedural-groom-density-mask
Procedural groom base density mask generation using region-based rules in Maya.

---

**Overview**

This project explores a procedural approach to generating base hair density masks
for XGen grooming in Maya, replacing repetitive manual mask painting.

---

**Pipeline Summary**

- Maya Python script extracts UV-aligned region data
- External Python script generates procedural base density masks
- Outputs are saved as UDIM-aligned texture maps for grooming workflows
